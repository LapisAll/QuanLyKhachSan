DỰ ÁN QUẢN LÝ KHÁCH SẠN (HOTEL MANAGEMENT SYSTEM)
📝 Giới thiệu chung
Dự án được xây dựng nhằm mục đích quản lý các hoạt động nghiệp vụ trong khách sạn, từ việc quản lý danh mục phòng, đặt phòng cho đến thanh toán hóa đơn. Đây là đồ án thực hiện trong quá trình học tập tại Trường Đại học Công nghiệp Việt - Hung.

🛠 Công nghệ sử dụng
Ngôn ngữ lập trình: C# (.NET Framework/Core)

Giao diện: WinForms

Kiến trúc: 3-Layer Architecture (Presentation - Business Logic - Data Access)

Cơ sở dữ liệu: SQL Server

Quản lý mã nguồn: Git & GitHub (Tuân thủ quy chuẩn Git-flow)

✨ Các tính năng chính
[x] Quản lý hệ thống: Đăng nhập, phân quyền người dùng (Admin/Nhân viên).

[x] Quản lý danh mục: Quản lý thông tin loại phòng, phòng, dịch vụ và khách hàng.

[x] Nghiệp vụ: Đặt phòng trực quan, nhận phòng và trả phòng.

[x] Thanh toán: Tính tiền dịch vụ, in hóa đơn cho khách hàng.

[ ] Báo cáo: Thống kê doanh thu theo tháng/năm (Đang phát triển).

🚀 Quy trình phát triển (Git-flow)
Dự án áp dụng quy chuẩn quản lý nhánh nghiêm ngặt:

Nhánh main: Chứa mã nguồn ổn định nhất.

Nhánh develop: Nhánh chính để tích hợp các tính năng.

Nhánh feature-*: Các nhánh riêng biệt để phát triển từng tính năng cụ thể.
